Digital Logic Control Unit (Virtual EV Transmission) — In Progress

Hardware-based state machine design for validating digital powertrain logic.

Objective

To design and validate a hard-wired digital state machine capable of simulating a 4-speed manual transmission for an electric vehicle testbed. This system validates the foundational logic of a "Virtual Gearbox" using discrete TTL components, decoupling critical state management from the main microcontroller software.

System Architecture

Discrete logic architecture:

  • Reference Input: Mechanical shift lever (SPDT momentary switches) representing driver intent.
  • Controller: 74LS-series TTL logic (discrete gates) — purely hardware-based state management.
  • State Machine: 74LS192 synchronous up/down counter tracking the "Virtual Gear" (N, 1, 2, 3, 4).
  • Visual Feedback: 74LS47 BCD-to-7-segment decoder driving a common anode LED display.
  • Safety Logic: 74LS08 (AND) and 74LS00 (NAND) gates implementing hard limits to prevent over-shifting.

Technical Implementation

Wave 1: Signal conditioning & safety logic

The core counter logic is supported by a robust "Wave 1" conditioning layer to ensure signal integrity and prevent illegal states:

  • 74LS14 (Hex Inverter with Schmitt Trigger): Conditions noisy mechanical switch signals. The Schmitt trigger's hysteresis eliminates contact bounce, converting jagged voltage spikes into clean, square-wave clock pulses for the counter.
  • 74LS08 (Quad 2-Input AND Gate): Implements the "Hard Stop" interlock. By AND-ing the "Shift Up" signal with the "Current State < 4" condition, this chip physically cuts the clock line when the limit is reached, preventing the counter from rolling over to zero.
  • 74LS00 (Quad 2-Input NAND Gate): Used for inverted logic and latching functions to stabilize state transitions during rapid shifting.
  • 74LS32 (Quad 2-Input OR Gate): Merges multiple fault conditions (e.g., limit reached OR emergency stop) into a single system-wide inhibit signal.

Hardware integration

Constructed a modular logic unit on a breadboard to operate independently of the flight controller. The system interfaces with the main vehicle via a 4-bit parallel binary bus, isolating "Decision Making" (Logic Unit) from "Actuation" (Microcontroller).

Power regulation

Dedicated LM2596 buck converter providing a stable, isolated 5.0V logic rail, protecting sensitive TTL gates from voltage sags caused by the main drive motor.
